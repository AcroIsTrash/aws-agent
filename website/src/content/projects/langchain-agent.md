### What it is

A from-scratch agent that takes a natural-language request, decides which tool to call, runs it, and returns a result. Building it meant getting hands-on with the pieces that make agentic systems work: prompt chaining, tool definitions, and the orchestration loop that ties a model to the actions it can take.

### Why I built it

Evaluating AI systems all day made me want to understand them from the inside — not just judge model outputs, but build the scaffolding that turns a language model into something that can actually *do* things. This agent is that hands-on counterpart to the evaluation work: the difference between grading the test and writing it.

It's also the workload that runs on top of the [AWS infrastructure project](/projects/#aws-ai-agent) — the two projects are two halves of the same story.
